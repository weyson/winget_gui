# Winget GUI - Менеджер обновлений Windows-приложений

**Другие языки：** [简体中文](README.md) | [繁體中文](README_zh_TW.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

Графическое приложение на Python с использованием tkinter для управления обновлениями Windows-приложений.

## Функциональные возможности

✅ **Графический интерфейс** - Интуитивно понятный и удобный интерфейс
✅ **Отображение списка приложений** - Показывает все приложения, требующие обновления, и информацию о версиях
✅ **Функция обновления списка** - Одно нажатие для получения актуального списка приложений, требующих обновления
✅ **Выделение всех/отдельных** - Гибкий выбор приложений для обновления
✅ **Пакетное обновление** - Одновременное обновление нескольких выбранных приложений
✅ **Индикатор прогресса** - Отображение прогресса и статуса обновления в реальном времени

## Системные требования

- Windows 10 или новее
- Python 3.7 или новее
- Windows Package Manager (winget)

## Установка

1. Клонируйте или скачайте этот проект
```bash
git clone <repository-url>
cd winget_gui
```

2. Убедитесь, что установлен Python 3.7+

3. Дополнительные зависимости не требуются, tkinter включен в стандартную библиотеку Python

## Использование

### Запуск приложения

```bash
python main.py
```

### Пошаговая инструкция

1. **Автоматическая проверка** - При запуске приложение автоматически проверяет доступность winget
2. **Автоматическое обновление списка** - Автоматически получает список приложений, требующих обновления
3. **Выбор приложений** - Нажмите на имя приложения, чтобы отметить его для обновления, или используйте кнопку "Выделить все"
4. **Обновление** - Нажмите кнопку "Обновить выбранные", чтобы начать обновление

## Описание функций

### Кнопка обновления списка
- Повторно сканирует систему и получает актуальный список приложений, требующих обновления
- Показывает общее количество приложений, требующих обновления

### Флажок "Выделить все"
- Отмечено: выбираются все приложения в списке
- Снято: снимается выделение со всех приложений

### Список приложений
Отображает следующую информацию:
- Имя приложения
- ID приложения (уникальный идентификатор)
- Текущая версия
- Доступная версия (последняя версия)
- Источник (winget, msstore и т.д.)

### Обновить выбранные
- Обновляются только выбранные приложения
- Перед обновлением требуется подтверждение
- Показывает прогресс и результаты обновления для каждого приложения

## Примечания

1. **Права администратора** - Некоторые обновления приложений могут потребовать прав администратора
2. **Сетевое подключение** - Для обновления приложений требуется интернет-соединение
3. **Установка winget** - Убедитесь, что Windows Package Manager установлен в системе
4. **Таймаут** - Таймаут для обновления одного приложения составляет 5 минут

## Устранение неполадок

### Winget не найден
- Сообщение об ошибке: `команда winget не найдена`
- Решение: Установите "App Installer" из Microsoft Store

### Проблемы с правами доступа
- Некоторые обновления приложений могут потребовать прав администратора
- Попробуйте запустить приложение от имени администратора

### Неудачное обновление
- Проверьте сетевое подключение
- Посмотрите подробное описание в сообщении об ошибке
- Попробуйте вручную запустить `winget upgrade <package-id>`
