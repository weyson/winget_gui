# Winget GUI - Windows 소프트웨어 업데이트 관리자

**다른 언어 버전：** [简体中文](README.md) | [繁體中文](README_zh_TW.md) | [English](README_en.md) | [日本語](README_ja.md) | [Русский](README_ru.md)

Python tkinter 기반의 그래픽 인터페이스 애플리케이션으로, Windows 소프트웨어 업데이트를 관리합니다.

## 기능 특징

✅ **그래픽 사용자 인터페이스** - 직관적이고 사용하기 쉬운 인터페이스
✅ **소프트웨어 목록 표시** - 업데이트 가능한 모든 소프트웨어와 버전 정보 표시
✅ **새로 고침 기능** - 업데이트 가능한 소프트웨어의 최신 목록을 한 번에 가져오기
✅ **전체 선택/단일 선택** - 업데이트할 소프트웨어를 유연하게 선택
✅ **일괄 업데이트** - 여러 개의 선택한 소프트웨어를 한 번에 업데이트
✅ **진행 상황 피드백** - 업데이트 진행 상황과 상태를 실시간으로 표시

## 시스템 요구 사항

- Windows 10 이상
- Python 3.7 이상
- Windows Package Manager (winget)

## 설치 방법

1. 이 프로젝트를 클론하거나 다운로드합니다.
```bash
git clone <repository-url>
cd winget_gui
```

2. Python 3.7+가 설치되어 있는지 확인합니다.

3. 추가적인 의존성 설치가 필요하지 않습니다. tkinter는 Python 표준 라이브러리에 포함되어 있습니다.

## 사용 방법

### 애플리케이션 시작

```bash
python main.py
```

### 작업 단계

1. **자동 확인** - 애플리케이션 시작 시 winget 사용 가능성을 자동으로 확인합니다.
2. **자동 새로 고침** - 업데이트 가능한 소프트웨어 목록을 자동으로 가져옵니다.
3. **소프트웨어 선택** - 소프트웨어 이름을 클릭하여 업데이트할 소프트웨어를 체크하거나 "전체 선택" 버튼을 사용합니다.
4. **업데이트** - "선택한 항목 업데이트" 버튼을 클릭하여 업데이트를 시작합니다.

## 기능 설명

### 새로 고침 버튼
- 시스템을 다시 스캔하여 최신 업데이트 가능한 소프트웨어 목록을 가져옵니다.
- 현재 업데이트 가능한 소프트웨어 총 개수를 표시합니다.

### 전체 선택 체크박스
- 체크: 목록의 모든 소프트웨어 선택
- 취소: 모든 선택 취소

### 소프트웨어 목록
다음 정보를 표시합니다:
- 소프트웨어 이름
- 소프트웨어 ID (고유 식별자)
- 현재 버전
- 사용 가능한 버전 (최신 버전)
- 소스 (winget, msstore 등)

### 선택한 항목 업데이트
- 선택한 소프트웨어만 업데이트합니다.
- 업데이트 전에 확인이 필요합니다.
- 각 소프트웨어의 업데이트 진행 상황과 결과를 표시합니다.

## 주의 사항

1. **관리자 권한** - 일부 소프트웨어 업데이트에는 관리자 권한이 필요할 수 있습니다.
2. **네트워크 연결** - 소프트웨어 업데이트에 네트워크 연결이 필요합니다.
3. **Winget 설치** - Windows Package Manager가 설치되어 있는지 확인합니다.
4. **시간 제한 설정** - 단일 소프트웨어 업데이트 시간 제한은 5분입니다.

## 문제 해결

### Winget을 찾을 수 없음
- 오류 메시지: `winget 명령을 찾을 수 없습니다`
- 해결 방법: Microsoft Store에서 "App Installer"를 설치합니다.

### 권한 문제
- 일부 소프트웨어 업데이트에는 관리자 권한이 필요할 수 있습니다.
- 애플리케이션을 관리자 권한으로 실행해 보세요.

### 업데이트 실패
- 네트워크 연결을 확인합니다.
- 오류 메시지의 상세 설명을 확인합니다.
- `winget upgrade <package-id>`를 수동으로 실행해 보세요.
