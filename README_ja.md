# Winget GUI - Windows ソフトウェア更新マネージャー

**他の言語版：** [简体中文](README.md) | [繁體中文](README_zh_TW.md) | [English](README_en.md) | [한국어](README_ko.md) | [Русский](README_ru.md)

Python tkinter ベースのグラフィカルインターフェースアプリケーションで、Windows ソフトウェアの更新を管理します。

## 機能特徴

✅ **グラフィカルユーザーインターフェース** - 直感的で使いやすいインターフェース
✅ **ソフトウェアリスト表示** - 更新可能なすべてのソフトウェアとそのバージョン情報を表示
✅ **リフレッシュ機能** - 最新の更新可能ソフトウェアリストをワンクリックで取得
✅ **全選択/単選択** - 更新するソフトウェアを柔軟に選択
✅ **一括更新** - 複数選択したソフトウェアを一度に更新
✅ **進度フィードバック** - 更新進度と状態をリアルタイムで表示

## システム要件

- Windows 10 またはそれ以降
- Python 3.7 またはそれ以降
- Windows Package Manager (winget)

## インストール

1. このプロジェクトをクローンまたはダウンロード
```bash
git clone <repository-url>
cd winget_gui
```

2. Python 3.7+ がインストールされていることを確認

3. 追加の依存関係は不要です。tkinter は Python 標準ライブラリに含まれています

## 使用方法

### アプリケーションの起動

```bash
python main.py
```

### 操作手順

1. **自動チェック** - アプリ起動時に winget の可用性を自動的にチェック
2. **自動リフレッシュ** - 更新可能なソフトウェアリストを自動的に取得
3. **ソフトウェアの選択** - ソフトウェア名をクリックして更新するソフトウェアをチェック、または「全選択」ボタンを使用
4. **更新** - 「選択したものを更新」ボタンをクリックして更新開始

## 機能説明

### リフレッシュボタン
- システムを再スキャンし、最新の更新可能ソフトウェアリストを取得
- 現在の更新可能ソフトウェアの総数を表示

### 全選択チェックボックス
- チェック：リスト内のすべてのソフトウェアを選択
- 取消：すべての選択を解除

### ソフトウェアリスト
以下の情報を表示：
- ソフトウェア名
- ソフトウェアID（一意の識別子）
- 現在のバージョン
- 使用可能なバージョン（最新バージョン）
- ソース（winget、msstore など）

### 選択したものを更新
- 選択されたソフトウェアのみを更新
- 更新前に確認が必要
- 各ソフトウェアの更新進度と結果を表示

## 注意事項

1. **管理者権限** - 一部のソフトウェア更新には管理者権限が必要な場合があります
2. **ネットワーク接続** - ソフトウェアの更新にはネットワーク接続が必要です
3. **Winget のインストール** - Windows Package Manager がインストールされていることを確認してください
4. **タイムアウト設定** - 単一のソフトウェア更新のタイムアウト時間は5分です

## トラブルシューティング

### Winget が見つからない
- エラーメッセージ：`winget コマンドが見つかりません`
- 解決策：Microsoft Store から「App Installer」をインストール

### 権限の問題
- 一部のソフトウェア更新には管理者権限が必要な場合があります
- アプリケーションを管理者として実行してみてください

### 更新に失敗した場合
- ネットワーク接続を確認
- エラーメッセージの詳細を確認
- `winget upgrade <package-id>` を手動で実行してみてください
